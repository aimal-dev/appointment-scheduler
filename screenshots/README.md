# Screenshots Directory

This folder contains screenshots of the Appointment Scheduler plugin.

## Required Screenshots

Please add the following screenshot images to this folder:

1. **appointment-scheduler-main.png**
   - Screenshot of the main appointment scheduling interface
   - Should show the calendar on the left and time slots on the right
   - Recommended size: 1200x800px or larger

2. **appointment-scheduler-settings.png**
   - Screenshot of the admin settings page
   - Should show all settings options (time, interval, emails, timezone)
   - Recommended size: 1200x800px or larger

3. **appointment-scheduler-bookings.png**
   - Screenshot of the admin bookings dashboard
   - Should show the list of all appointments
   - Recommended size: 1200x800px or larger

## How to Take Screenshots

1. Open your WordPress site with the plugin activated
2. Navigate to the page with the appointment scheduler
3. Take a screenshot (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
4. Save the image with the exact filename mentioned above
5. Place it in this `screenshots` folder

## Image Format

- Format: PNG (recommended) or JPG
- Naming: Use exact filenames as listed above
- Quality: High resolution for better display in README
